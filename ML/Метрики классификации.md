# accuracy
$$\displaylines{accuracy = \dfrac{\text{количестчво верных}}{\text{общее количество}}}$$

$$\displaylines{precision = \dfrac{TP}{TP+FP} ~~-\text{ какой процент единичек настоящие единички (точность)}\\
recall = \dfrac{TP}{TP+FN} ~~-\text{ какой процент из всех единичек мы угадали (отзыв, полнота)}\\

F_{1}= \dfrac{precision~\cdot~recall}{precision~+~recall}


}$$

Для каждого объекта предсказали вероятность, если вероятность для объекта больше чем порог, то ставим ему 1, если меньше, то 0.

При пороге 0, True Positive = 1, False Positive = 1 (пометили все объекты как единички)
При пороге 1, True Positive = 0, False Positive = 0 (пометили все объекты как нули)
