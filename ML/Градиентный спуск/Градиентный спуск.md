### Градиентный спуск

**Градиентный спуск** — это итерационный метод оптимизации, используемый для поиска локального минимума (или максимума) функции. Мы хотим найти как можно более хороший локальный минимум, а в идеале глобальный минимум. Основная идея метода заключается в следующем:

- **Вычисление градиента:**  
    В каждой точке функция имеет градиент, который указывает направление наибыстрейшего возрастания значения функции. Для поиска минимума используется отрицательное направление градиента.
    
- **Итерационное обновление параметров:**  
    Начинаем с некоторой стартовой точки x0 и на каждом шаге обновляем значение по правилу:


![[Pasted image 20250204173937.png]]

**Цель метода:**  
    Цель — последовательно уменьшать значение функции, двигаясь в направлении наибыстрейшего уменьшения, пока не будет достигнут локальный минимум или пока не выполнится условие остановки.


**Важный момент для понимания**
По сути мы просто из текущей точки Xk - вычитаем скорость изменения текущей функции в данной точке (т.к у нас градиент, то он помогает работать с разными размерностясми пространства) и получаем следующее значени точки Xk + 1 в поиске локального минимума..
