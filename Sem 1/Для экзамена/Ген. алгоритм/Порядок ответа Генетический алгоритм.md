План: 
1. Когда полезен генетический алгоритм?
	1. Когда не можем построить градиент
	2. Когда нужна многопараметрическая оптимизация (можно использовать несколько градиентных функций, но там нужно думать)
2. Многопараметрическая оптимизация. Доминантность и оптимальность по Парето
	1. Пример с такси и автобусами
	2. Многопараметрическая оптимизация (что на выходе функции, которую мы оптимизируем?)
	3. Доминантность по Парето
	4. Оптимальность по Парето (если что посмотреть)
3. Общая идея генетического алгоритма
	1. Идея: метод безградиентной оптимизации, имитирует эволюцию
	2. Основные этапы и идея генетического алгоритма (посмотреть)
4. Особь. Представление
	1. Что такое особь?
	2. Какие требования для особи? (если что посмотреть)
	3. Что такое популяция?
	4. От чего зависит выбор представления? (от задачи, чтобы выполнялись операции и получали особь)
5. Фитнесс-функция
	1. Кратко про фитнесс - функию (нужна для оптимизации), что определяет
	2. Фитнесс в случае однопараметрической оптимизации
	3. Фитнесс в случае многопараметрической оптимизации (посмотреть)
6. Методы селекции
	1. Элитизм (плюсы, минусы)
	2. Элитизм + случайность (плюсы по сравнению с элитизмом)
	3. Рулетка (недостаток, как исправить?)
	4. Турнир(без вероятностей, с вероятностями, недостаток)
	5. Stochastic Universal Sampling (шаги по прямой с интервалом преимущество по сравнению с рулеткой)
	6. Селекция с наследием (принцип наследия, для чего, типичная проблема, типичное решение)
7. Кроссовер
	1. Требования к кроссоверу 
	2. Одноточечный кроссовер
	3. Двухточечный кроссовер
	4. k-точечный кроссвовер
	5. Равномерный кроссовер
	6. Берем больше генов от приспособленной особи меньше от менее приспособленной
8. Мутации
	1. Что такое мутации?
	2. Для чего нужны мутации? Как они влияют на поиск решения?
	3. Чем полезна высокая мутация? Какие есть недостатки?
	4. Чем полезна низкая мутация? Какие есть недостатки?
	5. Как, скорее всего, имеет смысл регулировать мутацию?
	6. Виды мутаций: позиционные и обменом (посмотреть, уточнить в каких случаях они возможны)
9. Управление популяцией
	1. Сегрегация
		1. Что это?
		2. Для чего нужна?
		3. Виды сегрегации
			1. Острова
			2. Сетевые структуры (2D-сетка)
			3. Кластеризация
			4. Разбиение по полу (специализация)
			5. Разбиение по кастам
	2. Старение
		1. Что такое старение? (ограничение жизни)
		2. Для чего нужно старение? (борьба с отсутствием разнообразия, освобождение мест для новых особей)
		3. Реализации старения
			1. Особь живёт, размножается и умирает
			2. Особь живёт, ровно N итераций
			3. Зависимость возраста от качества решения (лучшие живут дольше)
			4. Старые сильнее мутируют, становятся лучше выживают, иначе погибают
		4. Проблема в старении? 
			1. Лучшие могут погибнуть
			2. Решение: зал славы
	3. Распараллеливание
			1. Ускорение вычислений
			2. Реализация: запускаем острова на разных компьютерах/потоках/процессорах, а потом объединяем
